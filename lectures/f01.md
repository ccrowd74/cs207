## High-level vs. low-level programming

This is math.
$$
f(a,b) = a + b
$$

This is high-level.

{% highlight python %}
def fun(a, b):
  return a + b
{% endhighlight %}

This is low-level.
```c
int32_t fun(int32_t a, int32_t b)
{
  return a + b
}
```
